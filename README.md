# QR Code Generator

A simple and responsive QR Code Generator built using **HTML, CSS, and JavaScript**. Users can instantly generate QR codes for text, links, and URLs.

## 🚀 Features

* Generate QR codes instantly
* Supports text and URLs
* Clean and responsive UI
* Simple and lightweight
* Beginner-friendly project

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📂 Project Structure

```
qr-code-generator/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## 📸 Preview

Enter any text or URL and click **Generate QR Code** to create a QR code instantly.

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/vanshikaarajput/QR-code-generator.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
